Projects

Mothra

Mothra is a project planning application inspired by JIRA. It is a full-stack application that uses JavaScript, jQuery, Node, Express, Handlebars, MySQL and Sequelize that was developed over 2 weeks by a 4 person group. Mothra allows users to create projects which are assigned to a team. A team consists of multiple users/employees. Users can also create tasks for a specific project. Each project consists of many tasks. Each task can be assigned to one user/employee, whose team is responsible for the project. Each task has state of (new, in-progress, complete, closed).

Devour the Burgers

Devour the Burgers is a a restaurant app that lets users input the names of burgers they'd like to eat. Whenever a user submits a burger's name, the app will display the burger on the left side of the page -- waiting to be devoured. Each burger in the waiting area also has a Devour it! button. When the user clicks it, the burger will move to the right side of the page to the Devoured Burgers List. The user has the option to change a devoured burger back to a burger that is ready to eat with the Make it Again! button. The app app stores every burger in a database, whether devoured or not.

Employee Tracker

This project is a command-line application that makes it easy for non-developers to view and interact with information stored in a database. With Employee Tracker the user manages a company's employees using an interface that works with NodeJS, express, inquirer, and MySQL. The user is prompted with a list of choices to view, add, update or delete the database, and the information is displayed to them in an easy to view table in the console.

Adopt-A-Pet

This project is an animal adoption search and pet resources mapping website that was developed by a 3-person group over the course of 2 weeks. It takes in user input to search the RescueGroups.org API for cats or dogs that are available for adoption from adoption agencies. It retrieves the data in a JSON format and dynamically updates the HTML and CSS via jQuery and JavaScript to display the animals available. It also offers the user resources like vets, pet stores, etc. using the Google Maps JavaScript API and Google Places library based on the user's location.

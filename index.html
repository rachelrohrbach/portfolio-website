Projects

Adopt-A-Pet

This project is an animal adoption search and pet resources mapping website. It takes in user input to search the RescueGroups.org API for cats or dogs that are available for adoption from adoption agencies. It retrieves the data in a JSON format and dynamically updates the HTML and CSS via jQuery and JavaScript to display the animals available. It also offers the user resources like vets, pet stores, etc. using the Google Maps JavaScript API and Google Places library based on the user's location.

Code Quiz

This project is a timed quiz that tests the user’s code-based knowledge. It calculates the user’s score by the time remaining. Answering quickly and correctly results in a higher score. Answering incorrectly results in a time penalty. When the timer runs out and/or all questions are answered, the user inputs their initials to save the score locally on a high score list with up to 10 of the highest scores.

Weather Dashboard

This project is a weather dashboard app with search functionality to find current weather conditions and forecasted weather for multiple cities. It uses the OpenWeather API to retrieve data for the cities, AJAX to hook into the API to retrieve the data in JSON format, dynamically updated HTML and CSS powered by jQuery, and displays a search history which the user can click to access past cities searched via local storage.
